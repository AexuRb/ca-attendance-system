# 本地运行说明

## 1. 启动网站

首次运行前，先把：

```text
local-config.example.bat
```

复制为：

```text
local-config.bat
```

然后修改里面的 MySQL 密码和初始管理员密码。

双击项目根目录下的：

```text
start.bat
```

脚本会自动启动后端服务，并打开：

```text
http://127.0.0.1:8080
```

如果提示 8080 端口已经有程序在运行，通常表示网站已经启动，可以直接访问上面的地址。

## 2. 备份数据

管理员或会长登录后台后，进入：

```text
维护
```

点击“一键备份”。备份文件会生成到：

```text
backups/app
```

文件名格式类似：

```text
backup_20260627_012457.zip
```

备份包内包含成员信息、签到记录、操作日志和值班星期配置，适合留档和迁移核对。

## 3. 修改本地配置

数据库账号、端口和初始管理员配置在：

```text
local-config.bat
```

如果 MySQL 密码或端口变化，只需要改这个文件。
