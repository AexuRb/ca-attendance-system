# 本地运行说明

## 1. 启动网站

首次运行前，先把：

```text
local-config.example.bat
```

复制为：

```text
local-config.bat
```

然后修改里面的 MySQL 密码和初始管理员密码。

双击项目根目录下的：

```text
start.bat
```

脚本会自动启动后端服务，并打开：

```text
http://127.0.0.1:8080
```

如果提示 8080 端口已经有程序在运行，通常表示网站已经启动，可以直接访问上面的地址。

## 2. 备份数据库

双击：

```text
backup.bat
```

备份文件会生成到：

```text
backups
```

文件名格式类似：

```text
ca_attendance_20260624_125657.sql
```

## 3. 恢复数据库

可以把 `.sql` 备份文件拖到：

```text
restore.bat
```

也可以直接双击 `restore.bat`，再按提示输入备份文件路径。

恢复会覆盖当前数据库内容，执行前脚本会要求确认。

建议恢复前先运行一次 `backup.bat`，给当前数据库再留一份备份。

## 4. 修改本地配置

数据库账号、端口和初始管理员配置在：

```text
local-config.bat
```

如果 MySQL 密码或端口变化，只需要改这个文件。
