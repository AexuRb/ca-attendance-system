# 本地运行说明

## 1. 启动网站

首次运行前，先把：

```text
local-config.example.bat
```

复制为：

```text
local-config.bat
```

然后修改里面的 MySQL 密码和初始管理员密码。
默认 `APP_HOST=127.0.0.1`，表示网站只在当前电脑本机访问。

如果直接运行 `start.bat` 时还没有 `local-config.bat`，脚本会自动生成模板并停止。此时需要先打开 `local-config.bat`，把 `DB_PASSWORD` 改成当前电脑的 MySQL 密码，把 `INITIAL_ADMIN_PASSWORD` 改成初始管理员密码，再重新运行。

首次使用或换到新电脑时，先双击：

```text
init-db.bat
```

这个脚本会自动从项目当前位置读取 `database/schema.sql` 和 `database/seed.sql`，项目放在任意目录都不需要改 SQL 路径。

双击项目根目录下的：

```text
start.bat
```

脚本会自动启动后端服务，并打开：

```text
http://127.0.0.1:8080
```

如果提示 8080 端口已经有程序在运行，通常表示网站已经启动，可以直接访问上面的地址。

## 2. 备份数据

管理员或会长登录后台后，进入：

```text
数据中心
```

点击“一键备份”。也可以在“维护”页面执行同样操作。备份文件会生成到：

```text
backups/app
```

文件名格式类似：

```text
backup_20260627_012457.zip
```

备份包内包含成员信息、培训数据、排班数据、维修记录、签到记录、操作日志、值班星期配置和系统配置，适合留档和迁移核对。

## 3. 修改本地配置

网站监听地址、端口、数据库账号和初始管理员配置在：

```text
local-config.bat
```

如果 MySQL 密码或端口变化，只需要改这个文件。
